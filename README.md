# Projetos Front-End

Coleção de projetos desenvolvidos com **HTML, CSS e JavaScript**, com foco em prática de desenvolvimento web, interfaces responsivas e criatividade na construção de páginas.

## Sobre o Repositório

Este repositório reúne alguns projetos criados durante meus estudos e práticas na área de desenvolvimento front-end. Cada projeto foi desenvolvido com o objetivo de aprimorar conhecimentos em estruturação de páginas, estilização visual e interatividade com JavaScript.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Design Responsivo
- Organização de Layouts

## Projetos Disponíveis

### Conversor de Temperatura
Aplicação interativa que realiza conversão entre escalas de temperatura, utilizando JavaScript para cálculos e manipulação da interface.

### Loja Infantil
Página estilo e-commerce com exibição de produtos, imagens, preços e organização visual voltada para o público infantil.

### Hamburgueria
Landing page institucional para hamburgueria, com apresentação visual de produtos, identidade temática e layout atrativo.

### Página Tributo - 50 Cent
Projeto temático inspirado no artista 50 Cent, com foco em identidade visual, estruturação de conteúdo e apresentação artística.

## Objetivos do Projeto

- Praticar HTML, CSS e JavaScript
- Desenvolver interfaces modernas
- Melhorar responsividade
- Aprimorar organização de código
- Construir portfólio profissional
